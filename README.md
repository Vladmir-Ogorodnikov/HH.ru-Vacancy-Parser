# HH.ru Vacancy Parser

Парсер вакансий с hh.ru через официальное API.  
Собирает вакансии по заданным поисковым запросам и сохраняет данные в CSV-файл для дальнейшего анализа и визуализации.

---

## 🚀 Возможности

- Получение вакансий через API hh.ru
- Поддержка нескольких поисковых запросов
- Парсинг нескольких страниц выдачи
- Очистка HTML-тегов (`<highlighttext>`)
- Обработка отсутствующей информации о зарплате
- Сохранение данных в CSV-файл
- Подготовка данных для анализа в Pandas / Power BI / Tableau

---

## 🛠 Используемый стек

- Python 3.x
- requests
- pandas
- HH.ru API
